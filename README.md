## Convenciones del Proyecto

- Seguimos [Conventional Commits](https://www.conventionalcommits.org/) para mensajes claros y automáticos changelogs.
- Flujo GitFlow:
  - **main**: versiones estables (tags vX.Y.Z)
  - **develop**: integración de nuevas features
  - Features: desde develop → merge con merge commit
  - Release: desde develop → merge a main + tag
  - Hotfix: desde main → merge a main y develop
